API for ULV Weather

This api provides access to the live and past weather data recorded

Return to ULV Weather Home

Data is recorded and uploaded every 60 seconds Any data that shows a '-1' means there is an error. No result will show a '-1' otherwise
Wind direction shows where the wind is coming from ('N' means blowing from 'N', not to 'N')

List of api functions (http://api.ulvweather.com/[function name].php)
Function NameFunction Description
currentWeatherDataReturns most recent processed/averaged/summed data to be displayed
queryWeatherDataReturns most recent recording
lastHourWeatherDataReturns all recordings within the 60 minutes prior to the request (Approx. 60 rows)
lastHourWeatherAvgReturns an average of the last hours data in one row.
Some columns will return a sum, such as rainVolume
